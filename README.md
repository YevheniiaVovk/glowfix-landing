# 🛠️ GlowFix: Premium Appliance Repair Landing Page

Welcome! This is a high-end, responsive landing page built for professional appliance repair services. I created this project to demonstrate how modern UI/UI design can work for real-world service businesses.

## 🚀 Why this project?
In the service industry, **trust is everything**. This landing page isn't just a bunch of code—it's a tool designed to convert visitors into customers through:
* **Visual Trust:** Premium "Glassmorphism" effects and clean typography.
* **Smart UX:** A custom-built form with US phone masking to prevent user errors.
* **Speed:** Zero heavy frameworks. Just pure HTML and Tailwind CSS for lightning-fast loading.

## 💎 Key Highlights
* **Premium Feel:** Uses the *Plus Jakarta Sans* font for that "modern startup" look.
* **Mobile First:** Because 80% of people looking for a repairman are using their phones.
* **Validation Logic:** JavaScript that actually helps the user, showing clear errors if something goes wrong.
* **Authorized Service Focus:** Integrated sections for brand logos and service areas (ZIP codes).

## 🛠️ Tech I Used
* **HTML5** & **Tailwind CSS** (for the sleek look).
* **Vanilla JavaScript** (for the "brains" behind the form validation).
* **Google Fonts** (for that crisp typography).
